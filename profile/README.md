## Instalação do software

Instale o arquivo docker-compose.yml e rode o comando docker compose up -d

Automaticamente o front-end, back-end e banco de dados estaram prontos para uso

endereço do front-end: http://localhost:5173/login

endereço do back-end: http://localhost:8080/swagger
